我的Chrome浏览器都安装了哪些扩展插件

  • Larry
  • 2019-05-15
  • Chrome插件

大家好，我是独立开发者Larry，相比Safari我更喜欢用Chrome，除了开发方便调试，另外一个重要的原因就是Chrome 的好用除了它本身的流畅性以外，70% 的功劳更要归于其强大又顺手的扩展插件，这些插件满足了不同人的不同需求。

你有没有想过，你的浏览器能够自动帮你屏蔽无关的网站和广告，保持专注，今天这篇文章就为大家收集了 14 款 Chrome 插件，一起来看看。

Window Resizer - 窗口调整器

下载地址：https://chrome.google.com/webstore/detail/window-resizer/kkelicaakdanhinjdeammmilcgefonfh?hl=en

窗口调整器顾名思义是调整浏览器窗口大小的，通过它来调整浏览器窗口的大小以模拟各种屏幕分辨率，以便测试页面在不同浏览器分辨率上的布局，对设计和开发人员来说，都是必备的。

Live CSS Editor - 实时CSS编辑器

下载地址：https://chrome.google.com/webstore/detail/live-css-editor/oelggcmknbjmhkpgjfhakedcfnkgbdpg

实时CSS编辑器是一个简洁的浏览器扩展，使得在网站上试验CSS非常容易——在任何HTML页面上提供了一个文本框，并且可以完全控制站点，允许你可以自由编写CSS并立即查看结果。

Web Developer Checklist - Web开发人员核对表

下载地址：https://chrome.google.com/webstore/detail/web-developer-checklist/iahamcpedabephpcgkeikbclmaljebjp

Web开发人员核对表可分析任何网页是否存在违反最佳做法的情况，对于希望确保遵循最佳实践的Web开发人员来说，这个扩展程序能帮你轻松发现网站中的问题。

The Great Suspender - 伟大的吊杆

下载地址：https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg

The Great Suspender 是一个轻量级的扩展插件，当你同时打开多个标签，在可配置的时间长度之后未被查看的选项卡将在后台自动暂停，从而释放该选项卡占用的内存和CPU，减少Chrome的内存占用，使计算机顺利运行。

Screencastify - 屏幕录像机

下载地址：https://chrome.google.com/webstore/detail/screencastify-screen-vide/mmeijimgabbpbgpdklnllpncmdofkcpn

Screencastify是一个轻量级Chrome扩展程序，能够捕捉整个桌面、浏览器选项卡或网络摄像头录制流畅/高清视频，你可以在录制时使用外部麦克风或网络摄像头添加旁白，精简免费版支持录制视频时长10分钟，每月50个视频，付费后视频时长及数量无限制，并且还可以对录制好的视频进行修剪和编辑，支持合并多个视频、剪切/分割剪辑视频、导出为.mp4/动画.gif。

Adblocker for Chrome - NoAds

下载地址：https://chrome.google.com/webstore/detail/adblocker-for-chrome-noad/alplpnakfeabeiebipdmaenpmbgknjce

NoAds是一个5星广告拦截插件，提供了最完整的adblocker过滤列表，安装它可以阻止恶意软件、拦截广告、提升浏览速度，想必每一个人都会喜欢～

广告拦截- AdBlock

下载地址：https://chrome.google.com/webstore/detail/adblock/gighmmpiobklfepjocnamgkkbiglidom

AdBlock也是一个广告拦截插件，谷歌商店评星数是4.5星，拥有超过6000万用户，下载量超过2亿，能够阻止整个网络上的广告。

灯塔- Lighthouse

下载地址：https://chrome.google.com/webstore/detail/lighthouse/blipmdconlkpinefehnmjammfjpmpbjk

Lighthouse是一个开源的自动化工具，用于提高Web应用程序的性能、质量和正确性。在审核页面时，Lighthouse会针对该页面运行一系列测试，然后生成一个关于页面执行情况的报告。从这里，你可以使用失败的测试作为指标，了解你可以采取哪些措施来改进你的应用。

全页屏幕捕获- Full Page Screen Capture

下载地址：https://chrome.google.com/webstore/detail/full-page-screen-capture/fdpohaocaechififmbbbbbknoalclacl

获取当前浏览器窗口的完整页面截图的最简单方法。单击扩展图标（或按Alt + Shift + P），观看扩展捕获页面的每个部分，然后将其传输到屏幕截图的新选项卡，你可以将其作为图像或PDF下载，也可以直接将其拖拽到你的桌面。

Muzli 2 - 保持灵感

下载地址：https://chrome.google.com/webstore/detail/muzli-2-stay-inspired/glcipcfhmopcgidicgdociohdoicpdfc

Muzli是一个关于设计与灵感、技术与代码、新闻与商业、文化的扩展程序，安装该程序后，Muzli将取代你的默认选项卡成为你的主页，为你带来最新鲜的设计、技术、新闻、商业以及可以快速访问设计和技术网站中所有热门、新奇和创意的东西，满足你需要的所有设计灵感。

Restlet客户端 - REST API测试

下载地址：https://chrome.google.com/webstore/detail/restlet-client-rest-api-t/aejoelaoggembcahagimdiliamlcdmfm/related

Restlet Client是由开发人员为开发人员设计和开发的，能可视化地创建和运行单个HTTP请求以及复杂方案的程序，可以使直接的HTTP / REST资源可见，测试和测试自动化变得更加容易。

拾色器 - ColorZilla

下载地址：https://chrome.google.com/webstore/detail/colorzilla/bhlhnicpbhignbdhedgjhgdocnmhomnp

ColorZilla 是最受欢迎的Firefox开发者扩展程序之一，下载量超过500万，最终可供Chrome使用！使用ColorZilla，你可以从浏览器中的任何位置获取颜色读数，快速调整此颜色并将其粘贴到另一个程序中。

颜色标签 - Color Tab

下载地址：https://chrome.google.com/webstore/detail/color-tab/hchlgfaicmddilenlflajnmomalehbom

Color Tab 像是一个随机调色板，每打开一个新的标签或者刷新标签都能得到一个漂亮的配色方案。

页面标尺- Page Ruler Redux

下载地址：https://chrome.google.com/webstore/detail/page-ruler-redux/giejhjebcalaheckengmchjekofhhmal

Page Ruler Redux是一个核心的Web开发人员和设计工具，可以让你获得网页前端开发，网页设计或任何可能需要的网页元素的完美测量，以获得任何网页元素的像素尺寸和定位。
